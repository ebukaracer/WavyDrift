# WavyDrift
*A simple mobile flappy-style game I developed using Unity.*

## Game Logic:
*WavyDrift is a 3D side-scroller where the player attempts to fly between platforms without hitting them. Play the game to explore other features available!* 

## Tools Used:
* Unity 2020.3.10f1 
* Visual Studio (IDE)
* C# (Language)
* My [Major Packages](https://github.com/ebukaracer/Major_Packages/tree/main)

## Features:
- Object Pooling
- Save System
- Audio Manager
- InGame Shop
- Ads
